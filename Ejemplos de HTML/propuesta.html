Obras de arte

Se trata de una aplicacion en la cual el usuario lograra tener el conocimiento basico de las obras mas reconocidas en el mundo artistico



Funciones:

  1. Proporcionar datos curiosos de obras de arte

  2. Chat o foro para interactuar con otros usuarios

  3. Solicitar mas informacion sobre alguna obra de arte

  4. Seccion para subir tus propuestas de otras obras de arte
